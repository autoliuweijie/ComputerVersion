##0. 说明:

本项目基于opencv3, python2.7, 确保已经安装好opencv3, 且配置好python的site-packages, pkg-config。

1. c++：

    进入文件目录:

        # if you are in OS X, use clang++ instead of g++
        g++ $(pkg-config --cflags --libs opencv) main.cpp -o main
        ./main

2. python:

    进入文件目录:

        python **.py
